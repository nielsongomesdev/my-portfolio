# Nielson Gomes - Portfolio

Bem-vindo ao repositório do meu portfólio pessoal! Este projeto foi desenvolvido para apresentar meus projetos, habilidades e informações de contato com um visual agradável e minimalista.

## Tecnologias utilizadas

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS 4
- next-intl (internacionalização)
- Framer Motion (animações)
- Three.js + React Three Fiber + Drei (elementos 3D)
- tsParticles (efeitos de partículas)
- ESLint (qualidade de código)

## Como rodar o projeto localmente

1. Instale as dependências:
   ```bash
   npm install
   ```

2. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```

3. Acesse [https://portfolio-nielson.vercel.app/](https://portfolio-nielson.vercel.app/) no seu navegador para ver o resultado.
