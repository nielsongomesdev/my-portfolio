# Nielson Gomes - Portfolio

Bem-vindo ao repositório do meu portfólio pessoal! Este projeto foi desenvolvido para apresentar meus projetos, habilidades e informações de contato com um visual agradável e minimalista.

## Como rodar o projeto localmente

1. Instale as dependências:
   ```bash
   npm install
   ```

2. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```

3. Acesse [http://localhost:3000](http://localhost:3000) no seu navegador para ver o resultado.
